# Pro Temp LLC - Site Routes

This document lists all the routes created for the Pro Temp LLC website and their corresponding purposes.

## Generated Routes

### Main Pages
- `/` - Homepage (src/pages/index.astro)
  - Source: Root page with hero, services overview, testimonials, and contact
  - Features: Hero section, services grid, stats, testimonials, contact form

- `/services` - Services Page (src/pages/services.astro)
  - Source: Detailed HVAC services information
  - Features: Complete service listings, commercial HVAC, emergency service info

- `/about` - About Page (src/pages/about.astro)
  - Source: Company information and values
  - Features: Company story, values, certifications, service areas

- `/projects` - Projects Page (src/pages/projects.astro)
  - Source: Portfolio and case studies
  - Features: Project categories, featured projects, testimonials

- `/contact` - Contact Page (src/pages/contact.astro)
  - Source: Contact information and forms
  - Features: Contact form, business hours, location details, FAQ

## Route Structure

All routes are implemented as static pages using Astro's file-based routing system. Each page includes:

- Proper SEO meta tags
- Responsive design
- Consistent header and footer
- Call-to-action elements
- Professional styling

## Navigation

The main navigation includes all primary routes:
- Home → `/`
- Services → `/services`
- About → `/about`
- Projects → `/projects`
- Contact → `/contact`

- No blog content was detected in the original site, so no blog routes were created
